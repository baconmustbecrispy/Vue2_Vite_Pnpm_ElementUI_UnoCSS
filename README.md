# Vue2 + Vite + Pnpm + ElementUI + UnoCSS

## 環境

- node: 14.18.0
- npm: 8.17.0
- pnpm: 7.15.0
- vue2: 2.7.14
- element-ui: 2.15.10

> Node 版本必須 v14.18.0+ 或 v16.0.0+
> Vite uses node: import which is supported in v16.0.0+ and v14.18.0+.
> v15 does not support this. https://nodejs.org/api/esm.html#node-imports

## 執行

安裝 pnpm

```sh
npm i -g pnpm
```

安裝 deps

```sh
pnpm i
```

本地測試

```sh
pnpm dev
```
